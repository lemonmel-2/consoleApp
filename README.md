# Game Backend Console App

## Description
This is a C# console application that acts as a temporary backend for a frontend game. It provides basic functionality for authentication, item management, score recording, and leaderboard viewing. The database is currently using json file as temporary solution.

## Features
- **Authentication**: Login and Register users.
- **Item Management**: Generate items and add items to user inventory.
- **Score Recording**: Record user scores.
- **Leaderboard**: View the leaderboard based on user scores.
